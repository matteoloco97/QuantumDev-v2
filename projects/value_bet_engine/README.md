# Betting Analytics Engine

Modular ecosistema di raccolta e analisi dati per betting.

## Features

- Raccolta dati da API e web scraping
- Elaborazione dati con pandas e numpy
- Visualizzazioni avanzate con matplotlib
- Sistema modulare facile da estendere

## Requirements

- Python 3.9.x
- librerie specificate in requirements.txt
